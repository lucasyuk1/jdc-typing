
# JDC - Teste de Digitação (Starter)

Projeto inicial para um site de teste de digitação (subir no Vercel).

**O que tem aqui**
- Next.js (App usando pages)
- TailwindCSS (config pronta)
- Supabase client (exemplo)
- Páginas: login, cadastro, teste, ranking, admin
- Exemplo de tema neon inspirado nas imagens fornecidas

**Como usar**
1. Crie um projeto no Supabase e pegue `SUPABASE_URL` e `SUPABASE_ANON_KEY`.
2. Crie um arquivo `.env.local` com as variáveis:
   - SUPABASE_URL=your_url
   - SUPABASE_ANON_KEY=your_anon_key
3. No terminal (ou no Vercel) rode:
   - npm install
   - npm run dev

No Vercel, basta conectar o repositório e definir as variáveis de ambiente.
